Site Plan

Site Name

WorkNest

The name WorkNest combines work with nest, a place to settle, grow and build a career. It convenes a sense of belonging and opportunity, positioning the platform as a welcoming hub for job seekers The name is short, memorable, and brandable.

WDD130 WDD131 WDD231

Site Purpose

WorkNest is a responsive comma client-side job board web application that allows users to browse, search. and filter job listings. The site provides a streamlined experience for job seekers to discover opportunities, safe favorites, and submit applications.

The site also displays a real-time local weather location and a three day forecast on the home page using the OpenWeatherMap API and the visitor's geolocation, giving job seekers relevant context as they plan their day.

Back-end features such as user accounts and database storage are out of scope for this project and will be addressed in future courses

Scenarios

The following questions represent what a typical WorkNest visitor might ask:

"What jobs are currently available in my field?"
"How do I apply for a job i found on this site?"
"Can I save jobs am interested in an come back to them later?"
"what is the weather like in my area today?"

Color Scheme

The palette uses deep navy and indigo tones as the foundation, with a vivid purple accent for call to action, and sky blue for informational elements. All text on light colors uses dark navy to meet WCAG AA contrast standards.

Page Description

Home

index.html

Landing page featuring a hero section, local weather widget (OpenWeatherMap API + Geolocation), featured jobs highlights, and site navigation.

Job Directory

jobs.html

Display all jobs listings dynamically loaded from local JSON file. Includes search, filter by category/location, favorite toggle with local storage, and modal dialogs for job details.

Apply

apply.html

A validated HTML application form for submitting a job application. On submission, data is displayed on a confirmation/action page. featured jobs highlights, and site navigation.
